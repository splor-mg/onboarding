# Onboarding SPLOR-MG

Este repositório tem como objetivo centralizar e padronizar o processo de onboarding de novos integrantes da SPLOR-MG.

A partir de um *issue template* estruturado, buscamos:

* ✅ Organizar as etapas de estudo e ambientação técnica.
* ✅ Garantir o cumprimento das demandas administrativas e burocráticas.
* ✅ Promover boas práticas de documentação e colaboração no GitHub.
* ✅ Facilitar o acompanhamento do progresso do novoS integranteS.

Cada novo membro deve ter um *issue* próprio criado a partir do(s) template(s) de **"Onboarding"**, que funciona como checklist oficial do processo.

A proposta é simples: tornar o onboarding mais claro, rastreável, replicável e alinhado à cultura de documentação e trabalho colaborativo da equipe.
